Estilos

Los estilos nos permiten personalizar nuestras páginas web así como darle una apariencia más elegante. Para utilizarlas, deberemos añadir el atributo style a la etiqueta correspondiente de la siguiente manera: <etiqueta style="propiedad:valor;"> . Algunas propiedades son:

  • background-color para el color del fondo.
  • color para el color del texto.
  • font-family para cambiar las fuentes.
  • font-size para elegir el tamaño del texto.
  • text-alignment para modificar el alineamiento del texto.
  • width para escoger el ancho de un elemnto.

Antes de mostrar ejemplos, vamos a hacer una pequeña modificación al código para que la página se vea bien en cualquier tipo de dispositivo. Para ello hemos de añadir <meta name="viewport" content="width=device-width, initial-scale=1"> al inicio de nuestro archivo. Quedaría, por tanto, así:

  <!DOCTYPE html>
  <html lang="es-ES">
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Título de la página</title>

Los atributos, background-color y text-alignment y width ya los hemos utilizado con anterioridad. Veamos como funcionan los otros. Para ello utilizaremos la etiqueta span que nos permiten cambiar estilos dentro de otra etiqueta.

<p> En este párrafo hay letras <span style="color:Blue">verdes</span> y <span style="color:Green">azules</span>. Aunque los <span style="font-family:courier;">colores</span> <span style="font-size:120%;">no</span> correspondan a lo que está <span style="font-size:80%;">escrito</span>.</p>

En este párrafo hay letras verdes y azules. Aunque los colores no correspondan a lo que está escrito.

Hasta ahora hemos tenido que cambiar uno a uno los estilos de los elementos de nuestra página. Esto puede ser problemático, si tenemos un elemento que se repite de forma numerosa. Para solucionar este problema, podemos definir un estilo al comienzo del documento para que se aplique a todos los elementos iguales. Por ejemplo, si quisiéramos que todas las celdas de nuestras tablas fueran de color rojo con letras amarillas, añadiríamos lo siguiente al comienzo del documento:

  <!DOCTYPE html>
  <html lang="es-ES">
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Título de la página</title>
  <style>
    td {background-color: red; color:yellow;}
  </style>

De esta forma, podemos crear tablas personalizadas de manera mucho más sencilla.

  <table>
    <tr>
	   <td> Hola </td> <td>mundo </td>
	</tr>
  </table>
Hola mundo

En ocasiones, nos gustaría tener dos o más estilos para un solo elemento. Para esto, utilizaremos clases. Estas se colocan en la sección style al comienzo del archivo y tienen la estructura .nombreDeLaClase {Estilo de la clase}. Es importante poner el punto justo antes de definir el nombre. Para este ejemplo, crearemos dos una llamada bordeFino y otra llamada bordeGrueso.

  <!DOCTYPE html>
  <html lang="es-ES">
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Título de la página</title>
  <style>
    td {background-color: red; color:yellow;}
    .bordeFino {border: 5px solid black;}
    .bordeGrueso {border: 10px solid pink;}
  </style>

Estas clases pueden ser tan sencillas o tan complicadas como queramos.

Esto es una prueba del uso de clases.

En el ejemplo anterior, hemos utilizado una clase en una etiqueta span pero podríamos haberla utilizado en cualquier otra. Esto es una de las ventajas de las clases. Podemos reutilizarlas en todas las etiquetas que queramos con una única definición. Si nuestro fichero es grade y necesita muchas clases es más cómodo crear un fichero donde almacenarlas. Este tipo de fichero tiene extensión .css (Cascading Style Sheets). En nuestro caso, podríamos crear un fichero, en la misma carpeta, que se llame nuestroEstilo.css y que en su interior contenga:

     td {background-color: red; color:yellow;}
    .bordeFino {border: 5px solid black;}
    .bordeGrueso {border: 10px solid pink;}

Para relacionar nuestra página con el archivo CSS para poder utilizar el contenido que hemos definido en él borraremos el estilo que habíamos definido anteriormente y añadiremos un enlace al documento de estilos.

  <!DOCTYPE html>
  <html lang="es-ES">
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Título de la página</title>
  <link rel="stylesheet" href="nuestroEstilo.css">

Como curiosidad para esta web se hay utilizado un archivo CSS gratuito que está disponible en este enlace.

Escritura matemática

Para poder escribir símbolos y elementos matemáticos, no nos vale solamente utilizar lo que hemos visto. Necesitamos recurrir a una biblioteca denominada MathJax. Por extensión, no vamos a cubir aqui todos detalles de la misma, pero sí como añadirla a nuestra págia para poder usarla. Para ello, debemos modificar nuevamente la cabecera de nuestro archivo.

  <!DOCTYPE html>
  <html lang="es-ES">
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Título de la página</title>
  <link rel="stylesheet" href="nuestroEstilo.css">
  <script type="text/x-mathjax-config">
MathJax.Hub.Config({
  tex2jax: {
    inlineMath: [['$','$'], ['\\(','\\)']],
    processEscapes: true
  }
});
MathJax.Hub.Config({
  TeX: {
    Macros: {
      R: "{\\mathbb R}",
      N: "{\\mathbb N}",
      Z: "{\\mathbb Z}",
      bold: ["{\\bf #1}",1]
    }
  }
});
MathJax.Hub.Config({
  TeX: { equationNumbers: { autoNumber: "AMS" } }
});
TeX: {
  extensions: ["AMSmath.js", "AMSsymbols.js", "color.js"]
}
</script>
<script src='https://cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.4/MathJax.js?config=TeX-AMS_HTML' async></script>
