Creación de listas y tablas

Listas desordenadas

Para crear una lista donde los elementos no se enumeren usamos la etiqueta ul. Además cada ítem tiene que marcarse con la etiqueta li.

 <ul>
      <li> Elemento 1 </li>
      <li> Elemento 2 </li>
      <li> Elemento 3 </li>
   </ul>
  • Elemento 1
  • Elemento 2
  • Elemento 3

Podemos personalizar estas listas de distintas maneras.

 <ul style="list-style-type:circle;">
      <li> Elemento 1 </li>
      <li> Elemento 2 </li>
      <li> Elemento 3 </li>
   </ul>
  • Elemento 1
  • Elemento 2
  • Elemento 3 
 <ul style="list-style-type:square;">
      <li> Elemento 1 </li>
      <li> Elemento 2 </li>
      <li> Elemento 3 </li>
   </ul>
  • Elemento 1
  • Elemento 2
  • Elemento 3 
 <ul style="list-style-type:square;">
      <li style="background-color:green;"> Elemento 1 </li>
      <li style="background-color:white;"> Elemento 2 </li>
      <li style="background-color:green;"> Elemento 3 </li>
   </ul>
  • Elemento 1
  • Elemento 2
  • Elemento 3

Podemos también anidar listas unas dentro de otras. En ese caso, por defecto nos cambia la forma de las lista.

 <ul>
      <li> Elemento 1 </li>
      <li> Elemento 2 </li>
      <ul>
         <li> Elemento 2.1 </li>
         <li> Elemento 2.2 </li>
         <li> Elemento 2.3 </li>
      </ul>
      <li> Elemento 3 </li>
   </ul>
  • Elemento 1
  • Elemento 2
    • Elemento 2.1
    • Elemento 2.2
    • Elemento 2.3
  • Elemento 3

Listas ordenadas

De forma similar a lo que vimos anteriormente, podemos definir listas ordenadas con ol. Al igual que antes, necesitamos li para marcar cada ítem.

 <ol>
      <li> Elemento 1 </li>
      <li> Elemento 2 </li>
      <li> Elemento 3 </li>
   </ol>
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Veamos algunos ejemplos personalizados.

 <ol type="a">
      <li> Elemento 1 </li>
      <li> Elemento 2 </li>
      <li> Elemento 3 </li>
   </ol>
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3 
 <ol type="i">
      <li> Elemento 1 </li>
      <li> Elemento 2 </li>
      <li> Elemento 3 </li>
   </ol>
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3 
 <ol type="a" start="10">
      <li> Elemento 1 </li>
      <li> Elemento 2 </li>
      <li> Elemento 3 </li>
   </ol>
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3 
 <ol style="background-color:Yellow;">
      <li> Elemento 1 </li>
      <li> Elemento 2 </li>
      <li> Elemento 3 </li>
   </ol>
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

De igual forma podemos anidar listas unas dentro de otras.

 <ol>
      <li> Elemento 1 </li>
      <li> Elemento 2 </li>
      <ol>
         <li> Elemento 2.1 </li>
         <li> Elemento 2.2 </li>
         <li> Elemento 2.3 </li>
      </ol>
      <li> Elemento 3 </li>
   </ol>
  1. Elemento 1
  2. Elemento 2
    1. Elemento 2.1
    2. Elemento 2.2
    3. Elemento 2.3
  3. Elemento 3

Tablas

Las tablas son un poco más complejas de realizar. En primer lugar debemos utilizar la etiqueta table para marcar el inicio y el final de la msima. A continuación, tr para cada fila y, finalmente, td para cada celda.


 <table>
   <tr>
      <td>Nombre</td>
      <td>Edad</td>
      <td>Profesión</td>
   </tr>
   <tr>
      <td>Juan</td>
      <td>30</td>
      <td>Fontanero</td>
   </tr>
   <tr>
      <td>María</td>
      <td>35</td>
      <td>Astronauta</td>
   </tr>
   <tr>
      <td>Eva</td>
      <td>33</td>
      <td>Matemática</td>
   </tr>
 </table>
Nombre Edad Profesión
Juan 30 Fontanero
María 35 Astronauta
Eva 33 Matemática

Existe un tipo especial de fila que se utiliza para los encabezados. Su etiqueta es th. Con esto y algún ajuste en los estilos podemos hacer que nuestra tabla luzca mejor.


 <table style="width:100%; border:5px solid black; text-align: center;">
   <th>
      <td>Nombre</td>
      <td>Edad</td>
      <td>Profesión</td>
   </th>
   <tr style="background-color:lightgrey;">
      <td>Juan</td>
      <td>30</td>
      <td>Fontanero</td>
   </tr>
   <tr>
      <td>María</td>
      <td>35</td>
      <td>Astronauta</td>
   </tr>
   <tr style="background-color:lightgrey;">
      <td>Eva</td>
      <td>33</td>
      <td>Matemática</td>
   </tr>
 </table>
Nombre Edad Profesión
Juan 30 Fontanero
María 35 Astronauta
Eva 33 Matemática

También podemos unir columnas y filas utilizando colspan y rowspan, respectivamente. 


 <table style="width:100%; border:5px solid black; text-align: center;">
   <th>
      <td>Nombre</td>
      <td>Edad</td>
      <td>Profesión</td>
   </th>
   <tr style="background-color:lightgrey;">
      <td>Juan</td>
      <td colspan="2">Sin datos</td>
   </tr>
   <tr>
      <td rowspan="2">Sin datos</td>
      <td>35</td>
      <td>Astronauta</td>
   </tr>
   <tr style="background-color:lightgrey;">
      <td>33</td>
      <td>Matemática</td>
   </tr>
 </table>
Nombre Edad Profesión
Juan Sin datos
Sin datos 35 Astronauta
33 Matemática

Elementos de entrada

Los elementos de entrada son aquellos que nos permiten introducir información en la web. Existen numerosos elementos, de entre los cuales destacaremos los siguientes:

  • Campos de texto
  • Botones
  • Casillas verificables
  • Fechas
  • Botones de radio
  • Contraseñas

En esta sección no entraremos en los detalles de cada uno. Dejaremos eso para más adelante, cuando veamos el uso de JavaScript. Veremos sin embargo, algunos ejemplos de estos para acostumbrarnos a su estructura.

  <input type="text" name="campoTexto" value="" id="texto1">
 <br>
 <input type="button" name="boton" value="Púlsame" id="boton1">
 <br>
  <input type="checkbox" name="checkBox" value="Valor 1" id="checkbox1">
  <input type="checkbox" name="checkBox" value="Valor 2" id="checkbox2">
  <input type="checkbox" name="checkBox" value="Valor 3" id="checkbox3">
 <br>
  <input type="date" name="checkBox" value="" id="date1">
 <br>
  <input type="radio" name="radioButton" value="Valor 1" id="radio1">
  <input type="radio" name="radioButton" value="Valor 2" id="radio1">
  <input type="radio" name="radioButton" value="Valor 3" id="radio1">
 <br>
  <input type="password" name="password" value="" id="password1">
