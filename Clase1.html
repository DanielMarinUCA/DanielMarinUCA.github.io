¿Qué es una página web?

Una página web puede definirse como un fichero electrónico accesible en internet que contiene información en forma de texto, vídeo, etc. En general, están escritas utilizando el lenguaje HTML (HyperText Markup Language). A continuación mostraremos podemos crear nuestra propia página web.

Fundamentos de HTML

Primeros pasos

Para crear un fichero HTML simplemente tenemos que crear un archivo con algún programa similar a Block de Notas, Gedit, Notepad++... y guardarlo con la extensión .html. Notemos que los programas como Microsoft Word no sirven para escribir este tipo de archivos. Por defecto, el archivo raíz de una web se llama index.

Comenzamos abriendo nuestro editor y, en un nuevo archivo, escribimos lo siguiente:

  <!DOCTYPE html>
  <html lang="es-ES">
  <meta charset="UTF-8">

  <title>Título de la página</title>

  <body>
     Cuerpo de la página
  </body>
  </html>
Veamos que significa cada línea.
  • <!DOCTYPE html> indica a nuestro navegador que el archivo es, efectivamene un fichero HTML. Si bien esta etiqueta no es obligatoria, su ausencia puede producir errores en la visualización y por tanto, es una buena práctica utilizarla.
  • <html lang="es-ES"> sirve para declarar el idioma en el que está escrito el fichero. Esto es importante tanto para la traducción de nuestra web por parte de los navegadores como para su lectura por medio de sintetizadores de voz.
  • <meta charset="UTF-8"> permite utilizar más caracteres que ACII, incluidos los emoticonos como 💗.
  • <title>Título de la página</title> crea el título de la página que aparece en el navegador. En este ejemplo, aparece "Título de la página".
  • <body> Cuerpo de la página </body> contiene la información que será mostrada en la web. En este caso, aparecería la frase "Cuerpo de la página".
  • </html> marca el fin de la página

Es interesante notar que cuando escribimos en HTML, nos encontramos con la siguiente estructura <etiqueta>texto</etiqueta>. Esto marca la apertura y el cierre de una etiqueta. Se aplica al texto que hay entre ellos. Esto lo hemos visto con las etiquetas title, body e incluso html. Otros ejemplos son strong que sirve para escribir en negrita e i que nos permite escribir en cursiva.

En ocasiones nos interesa escribir notas en el código que no queremos que aparezcan cuando el navegador muestre el fichero. Esto es lo que se conoce como comentarios. Se escriben de la siguiente manera: <!- Esto es un comentario ->. En este caso, no habría que cerrar ninguna etiqueta.

Normalmente querremos empezar nuetra página con un encabezado. Tenemos seis niveles disponibles según queramos títulos, subtítulos, subsubtítulos, etc.

  <h1>Nivel 1</h1>
  <h2>Nivel 2</h2>
  <h3>Nivel 3</h3>
  <h4>Nivel 4</h4>
  <h5>Nivel 5</h5>
  <h6>Nivel 6</h6>

Podemos personalizar nuestras cabeceras modificando los estilos.


<h1 style="background-color:Blue; color:White; border:5px solid Red;">Nivel 1</h1>

Cuando querramos escribir texto, podemos utilizar la etiqueta p que nos marca donde debe de comenzar y acabar un párrafo. También es útil conocer la etiqueta br que sirve para crear un salto de línea. Esta última no necesita cerrarse. Si necesitamos realizar una cita, usamos blockquote.

Para insertar una imagen en nuestra página utilizaremos <img src="nombre_fichero_imagen">. Es recomendable cuando empezamos, colocar la imagen en la misma carpeta donde tenemos el archivo HTML, en caso contrario en source= deberíamos indicar la dirección donde se encuentra. Veamos un ejemplo.


	<img src='foto.webp' alt="gatito" style="width:33%;">

Ya hemos hablado de para que sirve src y hemos visto que style nos sirve para modificar opciones de estilo. Expliquemos entonces la opción que falta: alt. En ocasiones nuestra imagen no se va a poder mostrar ya sea por el tipo de formato, el navegador que estemos usando o porque la dirección a la misma sea errónea. En estos casos, se mostrará el texto que apaece en alt. Esto nos puede servir para ayudarnos a depurar el código. Además, si el usuario utiliza un lector de contenido, este leerá dicho texto.

Finalmente, para realizar enlaces con otras páginas utilizaremos la etiqueta a.

 <a href="https://www.uca.es"> Pinche aquí para ir a la página de la Universidad de Cádiz </a>
Pinche aquí para ir a la página de la Universidad de Cádiz

Ejercicios propuestos

  1. Crea una página llamada index.html que sirva de presentación al curso. Debe de contener, al menos, siguientes elementos:
    1. Un elemento h1 donde aparezca el nombre de la asignatura.
    2. Tu nombre para poder evaluar el trabajo.
    3. Una breve introducción a la página.
    4. Un enlace a una segunda página que llamada ejercicio2.html. Este segundo archivo lo elaboraremos en un futuro ejercicio.
  2. Sube los archivos a algún servidor que permita su visualización como Github o Inifinity Free y enviame el enlace.
